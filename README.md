# 茉莉妈妈短剧工作台

一款专业的 AI 短剧创作工具，帮助创作者快速将创意转化为高质量短视频内容。

## 核心功能

### 🎬 短剧制作工作流
- 智能解析剧本，自动提取角色、场景与分镜结构
- 基于角色参考图维持角色一致性
- 支持关键帧、过渡视频、背景音乐与音效合成
- 输出适合主流视频平台发布的完整内容

### 🖼️ 无限画布创作
- 可视化节点编排，自由组织创作流程
- 文本、图片、视频节点灵活组合
- AI 助手引导，从剧本到成片一键生成
- 支持多人协作与版本管理

### 📝 图文说模式
- 自动识别章节与段落结构
- 为段落生成匹配的视觉提示词
- 并发生成图片、语音与字幕素材
- 组合为可直接发布的视频内容

### 🎨 提示词库
- 专业电影级提示词模板
- 覆盖文生图、图生图、图转视频全流程
- 包含镜头语言、灯光、色彩等专业参数
- 支持自定义提示词管理

### 📤 内容发布
- 支持主流视频平台一键发布
- 自动生成标题摘要与标签建议
- 批量发布与任务管理

## 适用场景

- 🎭 短剧、微电影制作
- 📖 小说、剧本影视化
- 🎥 AI 图文说、解说视频
- 📱 短视频批量创作
- 🎨 角色一致性要求高的创作

## 快速开始

1. 访问应用地址并登录
2. 创建新项目，导入或输入剧本
3. 使用短剧制作工作流完成创作
4. 导出视频或直接发布到平台
